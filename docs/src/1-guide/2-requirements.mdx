## GIT

Git é um sistema de controle de versão distribuído, projetado para lidar com projetos de todos os tamanhos, desde pequenos projetos pessoais até grandes projetos corporativos. Ele permite que você acompanhe as alterações feitas em seus arquivos ao longo do tempo, reverta para versões anteriores, crie ramificações para trabalhar em diferentes recursos de maneira isolada e mesclar essas alterações de volta à versão principal do projeto.

A instalação do Git pode variar dependendo do sistema operacional que você está usando. Aqui estão as instruções para algumas plataformas populares:

### Windows

1. Baixe o instalador mais recente do Git em https://git-scm.com/download/win.
2. Execute o instalador e siga as instruções na tela para concluir a instalação.

### MacOS

1. Baixe o instalador mais recente do Git em https://git-scm.com/download/mac.
2. Execute o arquivo DMG baixado.
3. Siga as instruções na tela para concluir a instalação.

### Linux (Ubuntu)

1. Abra o terminal e execute o seguinte comando:

```
sudo apt-get update
```

2. Em seguida, execute o seguinte comando para instalar o Git:

```
sudo apt-get install git
```

Depois de concluir a instalação, você pode verificar se o Git está instalado corretamente executando o seguinte comando no terminal:

```
git --version
```

Isso deve imprimir a versão do Git instalada no seu sistema.

## NodeJS

O Node.js é uma plataforma de desenvolvimento de software de código aberto que permite executar JavaScript no lado do servidor. Ele foi criado em 2009 por Ryan Dahl e é baseado na engine JavaScript V8 da Google. O Node.js permite que os desenvolvedores construam aplicativos escaláveis de alta performance usando JavaScript, uma linguagem comumente associada ao desenvolvimento de aplicativos front-end para web. Ele é amplamente utilizado para construir aplicações web, APIs, ferramentas de linha de comando, aplicativos de desktop, entre outras soluções.

Para instalar e configurar o Node.js em seu sistema, siga os seguintes passos:

1. Faça o download do instalador adequado para o seu sistema operacional no site oficial do Node.js: https://nodejs.org/en/download/

2. Abra o arquivo de instalação e siga as instruções para instalar o Node.js. Durante a instalação, você pode optar por aceitar as configurações padrão ou personalizá-las de acordo com suas necessidades.

3. Depois de concluir a instalação, abra um terminal ou prompt de comando e verifique se o Node.js foi instalado corretamente digitando o seguinte comando:

```
node -v
```

Isso deve retornar a versão do Node.js instalada em seu sistema.

## NPM

O npm (Node Package Manager) é um gerenciador de pacotes para o Node.js, ou seja, é uma ferramenta que ajuda a instalar, gerenciar e compartilhar bibliotecas de código JavaScript que podem ser utilizadas em projetos Node.js. Com o npm, é possível instalar pacotes de terceiros para incluir funcionalidades específicas em seu projeto, além de publicar e compartilhar seus próprios pacotes com a comunidade. O npm é uma ferramenta essencial para desenvolvedores Node.js, permitindo uma gestão mais eficiente e organizada das dependências de seus projetos.

Para instalar e configurar o NPM em seu sistema, siga os seguintes passos:

1. Verifique se o NPM está instalado em seu sistema, digite o seguinte comando:

```
npm -v
```

Isso deve retornar a versão do NPM instalada em seu sistema.

2. Para realizar download dos packages @hubert/\*, você precisa logar no npm, para isso basta instalar e executar os seguintes comandos:

```
npx vsts-npm-auth -config .npmrc
```

## VSCode

Visual Studio Code é um editor de código-fonte gratuito e de código aberto desenvolvido pela Microsoft. Ele é um dos editores de código mais populares entre desenvolvedores de software devido à sua grande quantidade de recursos e extensões disponíveis que aumentam a produtividade durante o processo de desenvolvimento. O VSCode suporta várias linguagens de programação e sistemas operacionais, além de possuir integração com controle de versão e depuração de código.

Para instalar o Visual Studio Code (VSCode) em um computador com o sistema operacional Windows, siga os seguintes passos:

1. Acesse o site oficial do VSCode em https://code.visualstudio.com/.
2. Clique no botão "Download for Windows".
3. O download do instalador do VSCode será iniciado automaticamente. Quando o download for concluído, execute o instalador.
4. Na janela de instalação, clique em "Avançar" para prosseguir.
5. Aceite os termos de uso e clique em "Avançar".
6. Escolha o diretório de instalação do VSCode e clique em "Avançar".
7. Selecione as opções adicionais que deseja instalar (recomenda-se manter as opções padrão) e clique em "Avançar".
8. Clique em "Instalar" para iniciar a instalação.
9. Aguarde a conclusão da instalação.
10. Clique em "Concluir" para finalizar a instalação.

Após a instalação, o VSCode estará pronto para ser usado.
