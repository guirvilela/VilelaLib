# I18n

Esta biblioteca contem códigos para adicionar internacionalização aos projetos, como também funções para formatar diferentes tipos de dados, como números, datas, moedas, entre outros.

## Como utilizar

```
npm i @hubert/i18n
```

### Exemplo:

```ts
import { I18nProvider, useI18nMessage } from '@hubert/i18n';
```
