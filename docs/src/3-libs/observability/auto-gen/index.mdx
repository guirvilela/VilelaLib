# Observability

Esta biblioteca contem e deve somente conter codigo relacionado a observability (analytics e metricas).

## Como utilizar

```
npm i @hubert/observability
```

Exemplo:

```javascript
import { AnalyticsProvider } from '@hubert/observability';
```
