# CLI

## Como utilizar

```
npm i -D @hubert/cli
```

Comandos disponiveis:

```
hubert-cli help
```

Gerar um projeto web completo e configurado:

```
hubert-cli generate web
```

Inicia o servidor de mock, lendo os arquivos json de alguma pasta (path):

```
hubert-cli mock ./src/mocks
```
